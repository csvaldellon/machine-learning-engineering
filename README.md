- To test:
    1. `cd qa_model/test/`
    2. `python set_context.py` (will prompt to choose between `local` vs `hosted`, choose `hosted` for "cloud-hosted")
    3. `python get_answer.py` (will prompt to choose between `local` vs `hosted`, choose `hosted` for "cloud-hosted")
